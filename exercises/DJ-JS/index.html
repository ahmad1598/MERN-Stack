The square's color will change as follows:

  • Blue when the mouse hovers over the square
  • Red when the mouse button is held down over the square
  • Yellow when the mouse button is let go over the square
  • Green when the mouse is double clicked in the square
  • Orange when the mouse scroll is used somewhere in the window

    • You can also :
    Type "r" on the keyboard for "red"
    Type "b" on the keyboard for "blue"
    Type "g" on the keyboard for "green"
    Type "y" on the keyboard for "yellow"
    Type "o" on the keyboard for "orange"
